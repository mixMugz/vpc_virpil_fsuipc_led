# LED control for VPC Virpil through FSUIPC7

Required: Full version of [FSUIPC7](http://www.fsuipc.com/) by Pete & John Dowson's

## Instalation

**Add to FSUIPC7.ini:**
```
[LuaFiles]
1=VPC_Base
2=VPC_LP_#2
3=VPC_LT_MT50CM3
4=VPC_RCA_MT50CM2

[Auto]
1=Lua VPC_RCA_MT50CM2
2=Lua VPC_LT_MT50CM3
3=Lua VPC_LP_#2
```
